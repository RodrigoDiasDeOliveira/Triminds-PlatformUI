# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful and inclusive environment for everyone.

Participation in this project should be a harassment-free experience regardless of:

- Age
- Disability
- Ethnicity
- Gender identity
- Nationality
- Race
- Religion
- Sexual orientation
- Technical experience

## Expected Behavior

- Be respectful.
- Be constructive.
- Assume good intentions.
- Welcome new contributors.
- Focus discussions on technical topics.
- Help improve the ecosystem.

## Unacceptable Behavior

Examples include:

- Personal attacks
- Harassment
- Discrimination
- Offensive language
- Spam
- Trolling

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Repeated violations may result in removal from the project.

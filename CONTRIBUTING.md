# Contributing

Thank you for your interest in contributing to Triminds PlatformUI.

## Philosophy

This project values:

- Simplicity
- Reusability
- Accessibility
- Performance
- Clean Architecture
- Strong Type Safety

## Pull Requests

Before opening a Pull Request:

- Keep changes focused.
- Follow the existing project architecture.
- Update documentation when necessary.
- Add or update tests whenever possible.
- Ensure all checks pass.

## Coding Standards

- TypeScript only.
- React Functional Components.
- Composition over inheritance.
- Accessibility first.
- Avoid unnecessary dependencies.

## Commit Messages

The project follows Conventional Commits.

Examples:

```text
feat(button): add loading state

fix(theme): resolve dark mode issue

docs(adr): add architecture decision
```

## Reporting Issues

Please include:

- Expected behavior
- Actual behavior
- Steps to reproduce
- Environment details

## Questions

Architecture discussions are welcome through GitHub Discussions or Issues.
